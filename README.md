# Hub Culture Code Repo

### Instructions for installation

<dl>
  <dt>Install composer</dt>
  <dd>
  
  ```
  composer install
  ```
  </dd>

  <dt>Install node_modules</dt>
  <dd>
  
  ```
  mpm i
  ```
  </dd>

  <dt>To run the template, set the permissions</dt>
  <dd>
  
  ```
  chmod 777 ./resources/cache
  ```
  </dd>
</dl>
